# 07-云训练资源选型与成本对比（已脱敏数据上云）

> **创建**：2026-05-25
> **用途**：HR 已脱敏 / TCM 公开数据 / POC 试点 等**非涉密内网**场景的训练资源选型
> **适用前提**：客户素材**已脱敏**或本属于公开数据，业务方书面确认可上云
> **与 06 关系**：[06-涉密内网部署指南](./06-涉密内网部署指南.md) 涉密客户内网采购 ←→ **本文档** 已脱敏上云训练
> **核心结论**：HR 已脱敏场景首选 **AutoDL**（¥1500-2500/5 周），备选**智星云**（更便宜）/ **阿里云 PAI**（合规升级）

## 一、适用场景

### 1.1 决策树（决定走 06 还是 07）

```
客户数据敏感度？
├── 涉密 / 未脱敏 / 客户合同要求数据不出本地
│   └── 走 06-涉密内网部署（采购硬件 ¥180-280 万）
│
├── 已脱敏 / 业务方书面同意上云
│   └── 走 07（本文档）：算力专营平台 / 大厂私有 VPC
│
└── 完全公开数据（如 TCM 古籍 / 中医执业医师题库）
    └── 走 07（本文档）：任意平台
```

### 1.2 HR 场景的适用性判断

HR 158 md 已脱敏（央企 / 省属国企脱敏案例），但仍属"客户内部参考材料"：

| 维度 | HR 已脱敏数据 |
|------|--------------|
| 是否含真实企业名称 | ❌ 已 mask |
| 是否含真实薪资数字 | ⚠️ 部分案例含范围（如"30-50 万"）|
| 是否含可识别个人信息 | ❌ 已脱敏 |
| 业务方是否书面同意上云 | **必须确认** |
| 客户合同是否限制 | **必须确认** |

**结论**：HR 已脱敏场景**符合 07 适用前提**，但启动前**必须**：
- 业务方书面确认"该数据可用于云端 POC 训练"
- 训练完销毁所有云端实例和数据
- 不上传**未公开**的客户原始合同 / 财报 / 内部数据

## 四、备选方案 1：智星云（便宜 60%）

### 4.1 适用情景

- 预算极敏感
- 可接受相对新平台的服务波动
- A100 80G 仅 **¥2.5/h**（vs AutoDL ¥6.68）→ HR 5 周可压到 ¥600-1000

### 4.2 关键差异 vs AutoDL

| 维度 | AutoDL | 智星云 |
|------|--------|--------|
| 价格 | ¥6.68/h A100 80G | ¥2.5/h A100 80G |
| 用户量 | 大（社区主流）| 中等 |
| 文档 | 极全 | 中 |
| 算力稳定性承诺 | 无明确 | **承诺波动 ≤ 1%** |
| 稳定性 | 高 | 待验证 |
| 团队 onboarding | 国内主流，教程多 | 需 1-2 天学习 |

### 4.3 实操建议

- **第一次用先开 1 小时 A100 实例做 smoke test**（成本约 ¥3）
- 跑通后再正式做训练
- 关键 checkpoint 多备份（避免抢占式被打断）

## 五、备选方案 2：阿里云 PAI-DLC（合规升级）

### 5.1 适用情景

- 业务方对 AutoDL 协议有顾虑（"仅科研用途" / "敏感数据用户全责"）
- 客户要求 **VPC 私有网络隔离** + **三级等保认证**
- 预算允许（贵 3-5×）

### 5.2 价格估算（HR 5 周）

| 项 | 估算 |
|----|------|
| A100 通用型实例（gn7e）| ¥34.7/h |
| 5 周训练约 60 卡时 | ¥2,082 |
| 数据迭代 × 2-3 | +¥2,000-4,000 |
| 存储 + 数据传输 | +¥500 |
| **总计** | **¥5,000-10,000** |

vs AutoDL ¥1,500-2,500，**贵 3-4×**，但获得：
- VPC 私有网络隔离
- 三级等保认证
- 完整合同 + SLA
- 阿里云大厂合规背书

### 5.3 PAI 关键优势

- **PAI-DLC**：分布式深度学习训练，对 70B+ 大模型友好（HR 9B 用不到，但 TCM 8 亿 token 可能用到）
- **PAI-EAS**：在线推理服务
- **PAI-Studio**：可视化建模
- 与阿里云生态（OSS / RDS / SLB）无缝集成
- VPC + 安全组 + RAM 权限完整

### 5.4 推荐场景

| 场景 | 是否走 PAI |
|------|----------|
| HR POC / 内部演示 | ❌ 太贵，AutoDL 够 |
| HR 客户商业项目（业务方要求私有云）| ✅ |
| HR 客户要求三级等保 | ✅ 必走 |
| TCM 大数据量 CPT | ✅ 可选（多机分布式生态成熟）|

## 七、不推荐的平台

| 平台 | 不推荐原因 |
|------|----------|
| **共绩算力**（¥0.76/h 4090）| 网吧+IDC 池化，稳定性差，训练中途可能掉资源；适合个人推理实验，不适合生产训练 |

| GPU 黑市（论坛 / 二手）| 安全风险（被植入挖矿木马 / 数据被窃）|

## 九、数据安全实操（脱敏数据上云的最佳实践）

### 9.1 脱敏前置 Checklist

- [ ] 真实企业名称替换为 `[央企-A]` / `[省属-B]`
- [ ] 真实人名替换为 `[张总]` / `[李部长]`
- [ ] 具体薪资数字模糊化（"127.5 万" → "100-150 万"）
- [ ] 身份证号 / 手机号 / 邮箱**全部 mask**（Presidio 跑两遍）
- [ ] 业务方书面确认"该版数据可用于云端训练"

### 9.2 训练时实操约束

- [ ] 不上传**原始客户合同 / 财报 / 内部规定**
- [ ] 不在云端实例**留账号密码 / API key**（用环境变量）
- [ ] 训练完 **24h 内销毁实例**（不保留过夜）
- [ ] checkpoint 下载到本地后**立即销毁云端副本**
- [ ] 不在云端实例**截图分享**到公司 IM

### 9.3 退出 Checklist（训练完）

- [ ] 所有 checkpoint 已下载到本地
- [ ] 所有训练数据已下载或确认销毁
- [ ] 所有云端网盘清空
- [ ] 所有云端实例已关停 + 释放（不只是关机）
- [ ] 账号余额转出（如有）
- [ ] 业务方书面确认"训练已完成、云端数据已清理"
