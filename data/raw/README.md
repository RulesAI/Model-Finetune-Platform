# data/raw/ — 原始素材

本目录通过**软链**接入客户素材，原始文件不在本项目 git 仓库中。

## 当前接入

| 软链 | 真实路径 | 内容 |
|------|---------|------|
| `HR-薪酬素材/` | `_HR/文档素材/已处理素材/` | 158 份国企薪酬 md（约 140 万 token） |

## 为什么用软链而不是硬复制

1. **数据源单一**：客户素材在 `_HR/` 持续维护，软链让本项目自动同步
2. **避免冗余**：节省 6 MB 磁盘 + 避免版本漂移
3. **强调"涉密数据不进 git"**：软链天然提示开发者这里只是引用

## 新增素材接入

```bash
ln -s /absolute/path/to/source data/raw/<scenario-name>
```

并在本 README 表格追加一行。
